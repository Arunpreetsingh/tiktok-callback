UniqueStore.us — TikTok Shop Automation

TikTok Shop

Automated Product Videos for Korean Skincare

This tool automates the creation and publishing of TikTok Shop product videos for UniqueStore.us — a Korean skincare shop. It uses the TikTok Content Posting API to publish short-form product videos directly to the store's TikTok account.

🎬 Video Creation

Automatically generates 30-second TikTok-format product videos with AI voiceover, captions, and product images.

🤖 AI Captions

Uses AI to generate engaging product captions, hooks, and voiceover scripts tailored for Korean skincare audiences.

📤 Auto Publishing

Uploads finished videos directly to the store's TikTok account via the Content Posting API with proper captions and hashtags.

🔒 Single Account

This tool is used exclusively for UniqueStore.us's own TikTok account. No third-party accounts are accessed or affected.

How TikTok API is Used

This application uses the following TikTok API scopes:

  • user.info.basic — to verify the authenticated account before publishing
  • video.publish — to post product videos directly to the store's TikTok profile
  • video.upload — to upload video files as drafts for review before publishing

All API calls are made from the store owner's private machine and only target the store owner's own TikTok account. No user data is collected, stored, or shared with third parties.
