TikTok Content Posting API

Automated Product Videos for TikTok Shop Sellers

ShopReel helps e-commerce sellers create and publish short-form product videos directly to their TikTok Shop accounts using the TikTok Content Posting API. Sellers connect their own TikTok account via OAuth and publish videos on their behalf.

🎬 AI Video Creation

Generates TikTok-format product videos with AI voiceover, animated captions, and Ken-Burns photo effects.

🔐 OAuth Login

Sellers authenticate with their own TikTok account via TikTok's official OAuth 2.0 flow. No passwords stored.

📤 Direct Publishing

Finished videos are uploaded and published to the authenticated seller's TikTok profile via the Content Posting API.

📝 Draft Review

Sellers can upload videos as drafts to review and edit in the TikTok app before making them public.

How TikTok API Scopes Are Used

ShopReel uses the following TikTok API scopes. Each scope is required for a specific step in the video publishing workflow:

user.info.basic

Used to retrieve the authenticated user's TikTok username and display name after OAuth login. This confirms which TikTok account has been connected before any video is published, and is shown to the user in the app dashboard.

video.upload

Used to upload a video file to TikTok as an inbox draft. The seller receives an in-app notification, reviews the video in the TikTok app, and chooses to post or discard it. This scope enables the "draft review" workflow where sellers approve content before it goes live.

video.publish

Used to directly publish an approved video to the authenticated seller's TikTok profile with a caption, hashtags, and privacy settings specified by the seller. This is the primary publishing action that makes the product video live on TikTok.

How the Integration Works

  1. The seller clicks "Connect TikTok Account" and is redirected to TikTok's OAuth authorization page.
  2. The seller grants permission for user.info.basic, video.upload, and video.publish scopes.
  3. TikTok redirects back to this page with an authorization code. The seller copies the URL and pastes it into ShopReel to complete authentication.
  4. ShopReel calls user.info.basic to confirm the connected account and display it to the seller.
  5. The seller selects products and ShopReel generates a product video with AI captions and voiceover.
  6. ShopReel uses video.upload to send the video as a draft to the seller's TikTok inbox for review, or uses video.publish to post it directly to their profile.
